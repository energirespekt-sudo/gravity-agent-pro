# Gravity Agent Pro

A fast-paced typing game with retro aesthetics and modern polish.

## Features

- **Active Word Focus** - Highlight only the word you're typing (no visual confusion)
- **Chiptune Music** - 8-bit background music loop
- **40 Levels** - Progressive difficulty
- **Boss Battles** - Special boss encounters with unique mechanics
- **Power-Ups** - Freeze and Nuke abilities
- **High Score Persistence** - Saves your best score
- **Live Letter Highlighting** - Real-time visual feedback
- **Professional Polish** - Danger zone indicator, enhanced HUD, premium sound effects

## Installation

```bash
npm install
npm run electron:dev
```

## Desktop Launcher

Use the `launch.sh` script or the `.desktop` file on Linux.

## Tech Stack

- Vanilla JavaScript + Web Audio API
- Vite (dev server)
- Electron (desktop app)
- CSS3 Animations

## Controls

- **Type words** to destroy falling objects
- **Enter** for panic clear (levels 1-20 only)
- **Click power-ups** to activate
